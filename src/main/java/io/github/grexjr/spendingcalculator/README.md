<h1>SPENDING CALCULATOR</h1>
<h3>BY: GAVIN MARCH</h3>

<p>
    This is a Spending Calculator project made from scratch with the Java programming language. It is
a CLI program that allows users to input expenses and incomes with associated dates and descriptions.
It also provides summary information about the user's account, including average income and expense
across the whole account, total income and expense, and a visual of all individual incomes and expenses.
The program saves the user's account in a .csv file format with all information about each item saved 
as a comma-delimited collection of the date, amount, and description.
</p>
